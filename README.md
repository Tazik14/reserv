## Reserv

Now supports Windows and Linux.
To run required Python 3 and Python pip.

> macOS is temporarily not supported.

#### How to run it
For Windows:
* Step 1
```sh
cd <project folder>
```

* Step 2
```sh
python startgui.pyw
```

For Linux, use:
```sh
python3 startgui.pyw
```
